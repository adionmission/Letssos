VIRTUAL MACHINE

VMware workstation 12 PRO

The VMware workstation provides you the oppournity to virtualise your system so that you can run two or more than two operating systems in your computer.Only the iso type image can be virtualise.So VMware workstation will be the best option.It is developed by Microsoft.


REQUIREMENTS


1.64-bit x86 Intel Core 2 Duo Processor or equivalent, AMD Athlon 64 FX Dual Core Processor or equivalent
2.1.3GHz or faster core speed
3.2GB RAM minimum/ 4GB RAM recommended
