ABOUT KALI LINUX

FEATURES

It is the first Open Source Android penetration testing platform for Nexus devices, created as a joint effort between the Kali community member “BinkyBear” and Offensive Security. It supports Wireless 802.11 frame injection, one-click MANA Evil Access Point setups, HID keyboard (Teensy like attacks), as well as Bad USB MITM attacks.


The logo of Kali Linux appears on the background in the TV series Mr. Robot (2015) in episode 5 of season one. The Kali Linux logo also appears again as wallpaper in scenes from episode 1, episode 9 and more prominently in the episode 6, season 2 of the same series, when it is used by two of the main characters, Angela and Darlene.


SUPPORT

Kali Linux is distributed in 32-bit and 64-bit images for use on hosts based on the x86 instruction set and as an image for the ARM architecture for use on the BeagleBoard computer and on Samsung's ARM Chromebook.

TOOLS

Kali Linux includes many well known security tools, including:

1. Nmap

a port scanner

2. Aircrack-ng

a software suite for penetration-testing wireless LANs

3. Kismet

network detector, packet sniffer, and intrusion detection system for 802.11 wireless LANs

4. Wireshark

a packet analyzer

5. Metasploit Framework

computer security that provides information about security vulnerabilities

6. Burp suite

web application security scanners

7. John the Ripper

password cracker

8. Social Engineering Toolkit

advanced attacks against the human element

9. Maltego

used for open-source intelligence and forensics

10. Ettercap

attacks on LAN

11. OWASP ZAP

web application security scanners

12. Armitage

a graphical cyber attack management tool
