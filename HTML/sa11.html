Specially for First Year Students

UBUNTU

ubuntu is a linux based operating system. It is the most famous operating system in the world and specially, it is very useful for IT infrastructure students.

KALI LINUX

It is the world's most advanced penetration testing platform.It is basically used for hacking.Kali Linux is preinstalled with over 300 penetration-testing programs.

ARCH LINUX

Arch linux focuses on elegance, code correctness and simplicity, and expects the user to be willing to make some effort to understand the system's operation.

DEBIAN

Debian is an open source operating system which in licensed under GNU and comes form the UNIX family but its kernel is linux based.Debian has access to online repositories that contain over 50,000 software packages.

LINUX MINT

Linux Mint is a community-driven Linux distribution based on Debian and Ubuntu that strives to be a "modern, elegant and comfortable operating system which is both powerful and easy to use.

VM-WARE WORKSTATION

Do you want to virtualise your computer then VMware workstation will be the best option due to which you can run different operating system on the same computer by using iso type file image.

RASPBIAN

The Raspberry Pi Foundation recommends the use of Raspbian, a Debian-based Linux operating system which comes with pre installed tools.

YOSEMITE

OS X Yosemite is the eleventh major release of OS X (now named macOS), Apple Inc.'s desktop and server operating system
